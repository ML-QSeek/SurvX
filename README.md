# SurvX

> *From constraints, cognition grows.*

---

## 它是什么

SurvX 是一个智能体生存框架。

在只有物理法则和客观后果的环境里，智能体通过"经历→因果归纳→方案推演→行动验证"的闭环，从零长出认知，在约束边界之间维持长期生存。


## 为什么做这个

做 SurvX 的出发点，首先是出于兴趣——**对探索生命和智慧底层原理的好奇**。

一个生命体如何在没有外部老师、没有奖励信号的情况下，仅仅通过与环境的交互，就长出适应性的行为和认知？这是我们在做的核心问题。

在这个基础之上，它也探索另一种决策方式：不是靠奖励函数优化，不是靠直接给出答案，而是靠约束和后果来驱动。

学术上，它研究的问题是：**在多约束条件下（物理、生理、认知），智能体如何求得一条长期的生存曲线？** 不是求最优解，不是最大化奖励函数，而是在约束边界之间持续存在、不崩溃。


## 怎么工作

SurvX 把"世界规则"和"智能体认知"彻底拆成两套系统。

**Field（世界规则）**：对智能体是只读的。它定义实体长什么样、有什么约束边界、每 tick 执行什么物理法则。智能体做任何动作，它只返回一个客观后果。不说话，不解释，不给建议。

**S-Ego（智能体认知）**：动态更新、版本可回溯。它从每一次"动作→后果"中建立因果记录——哪些因素影响了哪些结果，影响的方向和程度如何。这些记录不断归并，形成一张关系网络。下次遇到类似情况，它用这张网络推演各种方案，筛掉不可行的，在可行的里面选一个执行。

Field 和 S-Ego 运行时完全隔离。改物理规则不用动认知代码，改认知策略不用动世界配置。

**核心闭环**：反馈 → 归并关系 → 推演变化 → 叠加状态 → 进入下一轮。每 tick 执行一次，智能体的认知从经历中持续生长。

三层认知架构：本能层（硬兜底）→ 认知层（因果归纳）→ 符号层（语言随认知自然生长，完全本地运行）。三层之上没有统一指挥的"大脑"。外部只提供实体模板和物理法则，不参与决策。


## 现在能跑什么

**Demo：鲁滨逊在荒岛**

7 岁男孩，南太平洋荒岛，初始饥饿度 75，体力 70。没有说明书，没有任务提示，没有"下一步该做什么"的引导。

他推树，树晃了。他吃红果子，肚子疼了。他喝溪水，不饿了。

几次经历后，他的认知里开始出现因果条目——红果子→肚子疼，溪水→缓解饥饿。这些不是代码写好的，是他用自己的身体验证的。错误的代价是死亡，正确的回报是继续活着。

体力持续下降逼他行动，饥饿逼他尝试。约束围成的可行域里，他每 tick 都在寻找下一步。整个过程可回溯、可解释——每一帧状态、每一个决策都有记录。

**当前状态**：基础闭环已跑通。体力自然消耗至零触发终止，仪表盘可实时观测。
